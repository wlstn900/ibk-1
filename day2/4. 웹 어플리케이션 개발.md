# **3. 웹 어플리케이션 개발**

## **사전 설치 환경/도구**
- JDK v1.8 설치(Desktop)
- maven 설치(Desktop)
- vscode Extension

## **1. vscode에서 maven project 만들기**
1. vscode 실행
2. EXPLORER 내에서 MAVEN PROJECTS 탭 오픈
3. +버튼 클릭
4. springboot 검색
5. springboot-blank-archetype 선택
6. project 생성 완료 
(groupId : com.ibk.springsampleartifactId : ibk)
